## 🔬 Featured Projects

### 🛰️ PanSRGAN — Dual-Branch GAN for Satellite Super-Resolution
> *Deep Learning · Computer Vision · Remote Sensing*

A novel **dual-branch GAN** for 4× super-resolution of multispectral satellite imagery using high-resolution panchromatic (PAN) guidance — enabling improved land-cover classification and urban mapping.

- Designed **PAN/MS Fusion Blocks (PMFBs)** with spatial attention-guided residual injection
- Composite loss: L1 + VGG-19 perceptual + Spectral Angle Mapper (SAM) + RaGAN adversarial
- **Results:** +3.1 dB PSNR · +5–8% SSIM · SAM = 3.2° · +4.7% segmentation mIoU · +4.4% classification accuracy
- Trained on **NVIDIA A100 GPUs** via Param Shivay National Supercomputing Facility

`PyTorch` `GAN` `VGG-19` `Remote Sensing` `Gaofen-2 GID-15` `A100`
